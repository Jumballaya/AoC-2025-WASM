- [WebAssembly Toolkit](https://github.com/WebAssembly/wabt)

## Running the code

Start by installing the [WebAssembly Toolkit (wabt)](https://github.com/WebAssembly/wabt). This will have the program `wat2wasm` that is used to compile the `.wat` files into `.wasm`.

Next install with `npm install` and then run with `npm start`
